## ESBO - Frontend (Landing Page)

Interfaz de usuario y estructura visual principal para la plataforma de la Escuela Sistémica. Este proyecto representa la primera fase del desarrollo (Landing Page estática) enfocada en la presentación de servicios, clínica de salud mental y formación de excelencia.

## Tecnologías Utilizadas
Este proyecto fue construido utilizando herramientas modernas para garantizar escalabilidad y rendimiento:
* **Framework:** [React.js](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
* **Iconografía:** [Lucide React](https://lucide.dev/)
* **Despliegue:** [Vercel](https://vercel.com/)
